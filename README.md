# 🌍 SDG Country Profiler (World Bank API)

A Streamlit dashboard that retrieves Sustainable Development Goal–related indicators using the World Bank API.

## Features

- Live API integration (World Bank Open Data)
- Country selector
- SDG-to-indicator mapping
- Interactive Altair time-series visualisation
- Proper datetime axis formatting
- Cached API calls for performance

## Tech Stack

- Python
- Streamlit
- Pandas
- Altair
- World Bank API

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
